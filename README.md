## 运行
```
yarn install
npm run dev
```
## 项目结构
```
/build          // webpack配置
/config         // webpack配置
/src
  /api          // api请求
  /assets       // 放置图片，css，icon等资源文件
  /components   // 组件
  /router       // 路由
  /service      // 业务逻辑处理
  /store        // vuex管理数据状态
  /utils        // 工具，服务
  /views        // 容器组件，和router中导入的组件对应
  App.vue       // 根组件
  main.js       // 入口文件
/static         // 静态资源
```
